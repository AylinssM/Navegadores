Historia del navegador

Microsoft Edge

Microsoft Edge es un navegador web desarrollado por Microsoft. Fue lanzado por primera vez en 2015 y se ha convertido en el navegador predeterminado en los sistemas operativos Windows 10. Edge está diseñado para ofrecer una experiencia de navegación rápida, segura y compatible con las últimas tecnologías web.

Edge utiliza el motor de renderizado "EdgeHTML" en sus primeras versiones, pero a partir de 2020, Microsoft anunció que Edge se basaría en el motor de renderizado de Chromium, el mismo motor utilizado por Google Chrome. Este cambio permitió una mayor compatibilidad y rendimiento en la navegación web.

En cuanto al motor de JavaScript, Edge utiliza el mismo motor "V8" que Google Chrome, ya que ambos están basados en Chromium. Esto garantiza una ejecución eficiente y rápida de código JavaScript en el navegador.
